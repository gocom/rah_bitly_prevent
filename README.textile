h1. Prevent module for rah_bitly

p. "Packagist":https://packagist.org/packages/rah/rah_bitly_prevent | "rah_bitly":http://github.com/gocom/rah_bitly

p. This is an example plugin for rah_bitly showcasing extending. The plugin will prevent generating Bitly links for articles on section basis. The options can be set from Textpattern's native graphical Sections editor.

h2. Installing

p. Using "Composer":http://getcomposer.org:

bc. $ composer.phar require rah/rah_bitly_prevent
